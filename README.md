# szm
A simple bash script to open social media in browser from terminal.

# How to use?
1. Open your terminal
2. Use `mkdir -p ~/.local/bin`
3. Use `mv /path/to/szm.sh/here ~/.local/bin/`
4. Try to write `szm.sh` into your terminal and then `help` to see available sites. Also you can add your sites into the script.
